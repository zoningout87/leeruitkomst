Reflectie

Voor het programmeren ging CSS op de manier die voorgesteld werd heel goed. Ik snapte de codes heel goed. Javascript was een heel ander verhaal. Ik wilde dat op dezelfde manier oppakken, maar het ging gewoon niet. Ik heb voor sommige stukken veel hulp moeten vragen omdat ik er niet aan uitkwam en had ook telkens het gevoel dat ik met -1 achterstond. Ik vind het wel enorm leuk om te doen en wil er ook verder meegaan. Ik dacht wel dat het me wat gemakkelijker afging als CSS. Veel eerder aan moeten beginnen en ik ben weer terug naar de basis gegaan, zodat ik straks andermans code wel kan lezen.
