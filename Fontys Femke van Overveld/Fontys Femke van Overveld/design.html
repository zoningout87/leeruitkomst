Interactive Graphic Novel: End of the Apocalypse

Github

In 2012 ben ik begonnen met dit verhaal te illustreren waarin niet gesproken wordt. In 2014 heb ik dit verhaal afgemaakt, maar het was voor mijn gevoel nooit echt af en ik wist niet goed hoe ik het wilde uitgeven. Na een paar jaar heb ik besloten om er een interactief comic van te maken. Voor dit stuk heb ik een parallel effect gemaakt voor een stuk bos die erin voorkomt. Het is nog steeds een RGB bos, omdat ik er nog niet helemaal uit ben om heel de comic met SVG te maken of toch met PSD. Files. Voor deze scene wilde ik graag een bos illustreren waarbij je het gevoel krijgt dat je zelf door het bos heenloopt, die ik gemaakt heb met behulp van HTML en CSS.


Interactive Graphic Novel: Lust - Love - Hate

Github

Dit verhaal is nog steeds een work in process, maar de heartmonitor die ik in de achtergrond wil afspelen, is steeds hetzelfde gebleven. De bedoeling is dat hoe verder je in het verhaal gaat, de heartmonitor steeds meer richting flatline gaat. Ik wil het zo opzetten, dat er verschillende slides zijn en elke slide een javascript gedeelte krijgt, waar de SVG steeds een andere path krijgt. Het figuur dat je ziet, heb ik jaren geleden verzonden, en is van een gedeelte van verhalen waar alleen de kleuren rood, geel, oranje, zwart, grijs en wit wordt gebruikt.

Het was de bedoeling dat de website een verhaal vertelde over een legend die gaande is in de bossen van Springfield, die de auteur zelf bijhoudt. Het was de bedoeling dat de website zelf ook een soort van vervloekt was, maar helaas had ik toentertijd nog niet alle handigheidjes onder de knie om dit helemaal te maken. Er kwam net iets meer bij kijken dan dat ik kon. Later wil ik wel deze website bijwerken, hoe ik toen had bedacht.
